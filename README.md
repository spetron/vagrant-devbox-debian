# vagrant-devbox-debian
Vagrant PHP Development Box (Debian) Setup

This is a vagrant debian box setup for PHP development.

# Included:
 * Debian 8.4
 * PHP 7.0
 * Postgresql 9.x
 * MySQL 5.x
 * Apache 2.x
 
 
# How to use:
- git clone the project.
- modify vagrant.yaml file according to your needs.
- vagrant up
